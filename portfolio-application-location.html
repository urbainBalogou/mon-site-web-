Système de réservation de voitures en ligne

Plateforme web de location de voitures

Un projet complet de plateforme web de location de voitures, permettant aux utilisateurs de consulter les véhicules disponibles et de réserver en ligne.

Fonctionnalités clés :

  • Consultation des caractéristiques des véhicules (prix, type de carburant, nombre de sièges, etc.)
  • Réservation personnalisée avec sélection de dates, lieu de livraison, et options comme "avec ou sans chauffeur"
  • Gestion de l'authentification pour les utilisateurs (inscription, connexion, réservations personnelles)
  • Système de gestion dynamique pour les administrateurs (ajout de véhicules, mise à jour de la disponibilité, suivi des réservations)

Technologies utilisées :

Python Django
