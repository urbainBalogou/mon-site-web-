Site web pour un restaurant

Solution web pour la gestion de menus et la visibilité en ligne

J’ai réalisé ce projet web pour un client travaillant dans le domaine de la restauration.

L’objectif ? Permettre à ce restaurant de gagner en visibilité et de proposer son menu de pizzas aux utilisateurs.

Mon client, étant novice dans les nouvelles technologies, a la possibilité de modifier facilement son menu en ligne grâce à une interface d’administration sur-mesure que j’ai réalisée.

Ce projet simple mais fonctionnel a donné entière satisfaction à mon client.

Technologies utilisées :

HTML CSS Django Python Heroku
